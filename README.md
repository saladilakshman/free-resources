# Free Resources Hub

Welcome to Free Resources Hub, your one-stop destination for accessing a plethora of free resources related to various fields in science and technology. Whether you're a budding hacker, aspiring UI designer, or AI enthusiast, you'll find a treasure trove of valuable resources here.

## Features

- **Auto Theme Switching**: Enjoy a seamless experience with automatic switching between light and dark modes based on the time of the day.
- **Bookmark Functionality**: Save your favorite resources for later use with our bookmark feature.
- **Share Functionality**: Share valuable resource links effortlessly with others.
- **Category Filtering**: Easily filter resources based on different categories such as hacking, UI design, extensions, AI, and more.

## Technologies Used

- **ReactJS**: Built using ReactJS for efficient and dynamic user interfaces.
- **Material-UI**: Utilized Material-UI for designing beautiful and responsive UI components.
- **Framer Motion**: Integrated Framer Motion for smooth animations and transitions.

## Installation

To get started with Free Resources Hub locally, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/free-resources-hub.git`
2. Navigate to the project directory: `cd free-resources-hub`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
